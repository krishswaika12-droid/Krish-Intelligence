# 🚀 Krish AI

Krish AI is a futuristic dual-mode AI assistant built using React, Vite, Tailwind CSS, and Gemini AI.  
It features a unique **Hacker Mode** and **Normal Mode** with fully dynamic UI transitions and advanced cyberpunk-inspired design.

---

## 🌟 Features

- 🟢 Dual AI Personalities
  - Hacker Mode
  - Normal Assistant Mode

- 🎨 Dynamic UI Switching
  - Cyberpunk Hacker Interface
  - Modern Bright AI Interface

- 🤖 Gemini AI Integration

- 📁 File Upload Support

- 💬 Real-time Chat Interface

- ⚡ Smooth Animations & Effects

- 📱 Fully Responsive Design

- 🔗 Developer Footer Links
  - GitHub
  - LinkedIn
  - Gmail

- 🌐 Hosted on Vercel

---

## 🖥️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Gemini API
- JavaScript
- CSS Animations
